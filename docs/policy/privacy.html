政策概要

{AniInfo 致力於提供「新番資訊查閱」服務。本 App 不要求註冊帳號，也不會要求你提供姓名、電話、地址等個人資料。 本 App 透過橫幅廣告（Google AdMob／Google Mobile Ads SDK）維持營運，當廣告顯示時，第三方可能會依其服務需要處理部分裝置與使用互動資訊。

我們蒐集哪些資料

（A）App 功能本身

  • 收藏清單、主題色等設定：儲存在你的裝置本機（DataStore），用於提供個人化體驗與備份/匯入功能。

（B）廣告（Google Mobile Ads SDK / AdMob）

  • IP 位址（可能用於估算一般位置）
  • 使用者互動資訊（例如點擊、啟動、觀看等互動行為）
  • 診斷資訊（效能、當機/卡頓等與 SDK/APP 表現相關資料）
  • 裝置/帳戶識別碼（包含 Android Advertising ID、App Set ID 等）

我們如何使用資料

  • 提供與維持服務：顯示動畫資料、渲染圖片、提供搜尋/篩選與收藏功能。
  • 個人化：保存主題色、收藏清單（僅在本機）。
  • 廣告展示：顯示橫幅廣告並進行基本的投放/量測與防詐（由 Google Mobile Ads SDK 進行）。

資料分享與第三方服務

本 App 使用第三方服務：

  • Google AdMob（Google Mobile Ads SDK）：用於顯示橫幅廣告。

我們不會將你的收藏清單或設定上傳作為「使用者資料檔案」來建立個人帳號，也不會販售你的個人資料。

你也可以閱讀：第三方服務與廣告說明

你的選擇與控制（廣告/識別碼）

  • 你可以在 Android 系統設定中管理 Advertising ID（例如刪除或重設 Advertising ID）。
  • 你可以在 Google 的說明文件中了解更多廣告技術與控制方式。

（提醒：本段僅提供資訊指引，實際選項依你的 Android 版本與系統介面為準。）

資料保存與安全

  • 本機資料：收藏清單與主題色等設定保留於裝置本機。你可透過「匯出收藏備份/匯入收藏」自行管理。
  • 我們採取合理的技術與流程以降低資料遭未授權存取的風險；但任何系統都無法保證絕對安全。

兒童隱私

本 App 並非以兒童為主要對象。若你是監護人並認為兒童在未經同意下提供了不當資訊，請透過下方聯絡方式與我們聯繫（或使用 App 內回報功能）。

政策更新

我們可能因法規、功能調整或第三方服務更新而修訂本政策。更新後會同步更新本頁的「最後更新日」，必要時也可能在 App 內提示。
