\# 新番資訊 Android App



提供本季新番、動畫資料庫查詢、收藏與設定功能的 Android App。

不提供播放功能，僅提供資訊與外部連結。



\## 功能

\- 本季新番列表

\- 動畫資料庫（搜尋/篩選/排序 + 分頁）

\- 收藏（本機保存 + 批量載入）

\- 設定（主題色、收藏備份匯出/匯入、回報錯誤、隱私權政策、第三方服務與廣告）



\## 技術

\- Kotlin + Jetpack Compose + Material 3

\- Navigation Compose

\- DataStore（themeColor / favorites）

\- Retrofit/OkHttp（或 Ktor）

\- Coil（圖片）

\- AdMob Banner（僅橫幅）
